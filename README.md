# My GeoCities Style Website

Welcome to my personal website created in the nostalgic style of GeoCities. This website is hosted on GitHub Pages.

## Structure

- **index.html**: Main entry point of the website.
- **css/**: Contains the CSS file for styling.
- **assets/**: Contains images and scripts.
  - **images/**: Directory for image files.
  - **scripts/**: Directory for JavaScript files (if any).

Visit [jesstytam.github.io](https://jesstytam.github.io) to see the website in action.
